# 🏥 Hospital Appointment Data Analysis (SQL Project)

## 📌 Overview
This project demonstrates SQL-based data analysis on hospital appointment data to generate operational insights such as patient activity, departmental performance, and cost trends.

The aim is to simulate real-world NHS-style reporting using structured data analysis techniques.

---

## 🎯 Objectives
- Analyse patient appointment behaviour
- Identify high-value patients based on cost
- Measure departmental performance
- Extract latest appointment records per patient
- Rank and compare patient spending patterns

---

## 🛠️ Tools & Techniques
- SQL (PostgreSQL/MySQL compatible)
- JOIN operations
- GROUP BY & HAVING clauses
- Aggregate functions (SUM, COUNT, MAX)
- Window functions (ROW_NUMBER, RANK)
- Subqueries for filtering analytical results

---

## 📊 Key Business Insights

### 1. Department Revenue Analysis
Identified total revenue generated per department to understand workload and financial contribution.

### 2. Patient Activity Tracking
Calculated number of appointments per patient to assess engagement levels.

### 3. High-Value Patients
Ranked patients based on total spending to identify top contributors.

### 4. Latest Appointment Tracking
Extracted the most recent appointment per patient for operational monitoring.

### 5. Cost Ranking Analysis
Ranked appointments within each patient group to analyse cost distribution patterns.
